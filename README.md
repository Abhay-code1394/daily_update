# Daily News

A newspaper-style, real-time news site built with Express and a Claude-powered backend.

## Quick Start
1. npm install
2. copy .env.example to .env and add your Anthropic API key
3. npm run dev
4. Open http://localhost:3000

## Features
- Category-based news browsing
- Hero + sidebar + grid layout
- In-memory category cache
- Admin cache clearing endpoint
